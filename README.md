# Pathfinder

## [Website](https://erendn.github.io/pathfinder-js/)

## Features
* Pathfinding visualizer
* Random maze creation
* Checkpoints
* Algorithms:
  * Dijkstra
  * A*
  * Breadth-First Search (BFS)
  * Depth-First Search (DFS)
